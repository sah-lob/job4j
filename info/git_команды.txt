Команды Git.

git branch - посмотреть какие ветки есть в репозитории.
git branch name (name - имя) - создание нового имени.
git checkout name (name - имя) - переключить разработку на новою ветку.
git status - показывает какие файлы были изменены. 
git add - добавить файл в изменения для будущего коммита.
git commit -m «Комментарий.» - фиксация изменений в локальном репозитории. 
git push origin name - команда создает ветку name и вносит изменения.
